feathernote

a rich-text note taking web app

Ruby, Rails, JS, React, Redux, PostgresSQL

Live github

Feathernote is a web app inspired by evernote. Users can sign up for an account (or browse the guest account) and securely log in to their own personal digital note binder. Feathernote uses quill to allow users to edit their notes with color, font styles/sizes, bulleted lists, ordered lists, line orientation and more! Notes can be stored in notebooks for clean organization and tagged with key words for further filtering. This single page app is both elegant and functional.

MELODICode

a musical step sequencer and drum machine

JavaScript, jquery, tone.js, HTML, CSS

Live github

MELODICode is an online browser-based musical step sequencer. It behaves similarly software step sequencers seen in DAWs like FL studio, Ableton Live, etc. Each row of cells in the sequencer belongs to a note in a scale (MELODICode uses the pentatonic scale so most notes sound good together). Users can activate these cells on the sequencer's grid by clicking on them. Once cells are activated and a user hits play, MELODICode loops through the sequencer from left to right triggering and playing any of the notes of the active cells creating a melody.

DOMagic

a light-weight DOM manipulation library

JavaScript

github

This wizardry is an original take on a classic DOM manipulation library. Features include event handling, node manipulation, a document ready queue, and AJAX requests. DOMagic has methods like handler methods like on(), off() and other useful methods including: html(), addClass(), append(), attr() and more. The library makes manipulating the DOM easy and seamless.
